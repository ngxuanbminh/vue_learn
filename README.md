# vue-learn-crash 


## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the JSON server (required before running the webapp):

```bash
npm run server
```

3. Run the development server:

```bash
npm run dev
```

## Additional Commands

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
